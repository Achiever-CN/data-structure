# Data Structure
## Introduction
### 基本概念和术语
- **数据**
    > 是描述客观事物的符号，是计算机中可以操作的对象，是能被计算机识别，并输入给计算机处理的符号集合
  - **_可以输入到计算机中_**
  - **_能被计算机程序处理_**
- **数据元素** 
    > 是组成数据的、有一定意义的基本单位，在计算机中通常作为整体处理。 也被称为记录
-  **数据项**
    > 一个数据元素可以自若干个数据项组成，数据项是数据不可分割的最小单位
-  **数据对象**
    > 是性质相同的数据元素的集合，是数据的子集
